# 🚗 AI Parking Detection System

An AI-powered smart parking system that detects whether a parking space is **Free** or **Occupied** using image processing and deep learning.

---

## 📌 Overview

This project uses **Computer Vision** techniques along with **YOLO (You Only Look Once)** and **OpenCV** to analyze parking images and determine availability.

It is built as a **Flask web application** where users can upload an image and get real-time parking status.

---

## 🚀 Features

- 📷 Upload parking images
- 🤖 AI-based detection (YOLO + OpenCV)
- ✅ Detects parking status: **Free / Occupied**
- 🌐 Simple web interface using Flask
- ⚡ Fast and lightweight system

---

## 🛠️ Tech Stack

- Python
- Flask
- OpenCV
- NumPy
- YOLOv8 (Ultralytics)
